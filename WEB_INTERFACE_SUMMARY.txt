🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

SIMPLEST WAY - Docker with Local Files:

  1. Copy your JSON files to a directory:
     mkdir benchmarks
     cp before.json after.json benchmarks/

  2. Run the web server:
     docker run --rm -p 8000:80 \
       -v ./benchmarks:/var/www/runs \
       moodlehq/moodle-bench-diff:latest

  3. Open browser:
     http://localhost:8000

🔧 TECHNICAL DETAILS
═══════════════════════════════════════════════════════════════════════════════

Framework:        Symfony 6.4
Templating:       Twig
Styling:          Bootstrap 5
Charting Library: Chart.js (via Symfony UX)
Data Loading:     Local files or AWS S3

Key Components:
  • IndexController     - Handles filtering and display
  • ComparisonController - Handles comparisons
  • DatasetLoader       - Loads from local/S3
  • ChartBuilder        - Generates Chart.js configs
